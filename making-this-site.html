Making This Website

Hello everyone,

As someone deeply fascinated by web development, I recently embarked on a project that not only challenged my skills but also fulfilled a class requirement. I decided to create a website, which this blog now calls home. This journey was filled with learning, experimentation, and, ultimately, a lot of fun.

The Foundation: HTML, CSS, and a Hint of JavaScript

For this site, I harnessed the foundational web technologies: HTML and CSS, with just a sprinkle of JavaScript to add interactivity. My inspiration came from various sources, including other blogs and random websites, which influenced the design you see today.

HTML and CSS Deep Dive

The process involved a significant amount of relearning. I revisited the nuances of HTML and CSS, which empowered me to strategically place elements across the web page. From my Web-design class, I recalled lessons on flexbox, which became an integral part of the layout. The overall structure aims to blend the straightforward appeal of a landing page with the personal touch of a semi-professional portfolio.

Design and Planning

To kickstart the design phase, I created mood boards and gathered resources from across the web to crystallize my vision. Attached here is the PDF of my web-design project proposal, which delves deeper into my intentions and methodologies. This phase was particularly thrilling; it reminded me of my childhood days when I would pick fonts for school presentations—only this time, the stakes were a bit higher, and the ideas, much wilder.

Tools of the Trade

For development, I used:

  • Visual Studio Code: An editor that’s robust yet user-friendly for coding.
  • Chrome Dev Tools: Essential for debugging and visualizing CSS layouts.

For beginners, I highly recommend starting with these tools. They are instrumental in building a foundational understanding before moving on to more complex frameworks.

The Creation Process

Everything progressed smoothly, although I had to accelerate my timeline due to upcoming exams and other projects. Fortunately, I completed the website just in time and it turned out exactly as I had envisioned.

Final Thoughts

I am truly proud of the result. Building this site was a reminder of how much fun creating websites can be. Each project is a step forward in this ever-evolving journey. So, to all aspiring developers out there, start building. Even if the results aren't perfect initially, each line of code you write is a leap towards improvement.
